UX Guidelines

1. 游戏中操作优先：反馈在 120ms 内可见，禁止阻塞弹窗。
2. 主界面电竞极简，设置页轻毛玻璃，风格分层但不割裂。
3. 核心动作可盲操：播放/停止/搜索/切换模式都可键盘完成。
4. 异常显性化：快捷键冲突、文件缺失、设备离线都要可视反馈。
5. 文案短句化：避免长段文本，优先状态词与动作词。
